## 📖 About

**VJTI StudyZone** is a free and open educational platform built to help students of **Veermata Jijabai Technological Institute (VJTI)** access academic resources from a single place.

Finding notes, previous year question papers, textbooks, and study materials often becomes difficult because they're scattered across multiple platforms and drives. VJTI StudyZone aims to solve this problem by organizing everything in one clean, user-friendly website.

The goal of this project is simple:

> **Make learning resources easily accessible to every VJTI student.**

## 💻 Running Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/VJTI-StudyZone.git
```

Move into the project directory:

```bash
cd VJTI-StudyZone
```

Open the project:

```bash
open index.html
```

or simply use the **Live Server** extension in Visual Studio Code.

## ❤️ Why this project?

As students, we've often spent hours searching across WhatsApp groups, Google Drive folders, and seniors' messages just to find one set of notes or a previous year's paper.

VJTI StudyZone was created to bring everything together in one place, making academic resources more organized, accessible, and helpful for every student.
